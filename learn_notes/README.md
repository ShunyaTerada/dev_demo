# 学習ノート (Learning Notes)

このディレクトリは、Antigravityでのハンズオン学習を通じて得た知識を体系的に整理・保存するための場所です。
**「時系列（Chronological）」**と**「知識体系（Knowledge Base）」**の2つの軸で管理します。

## 📂 フォルダ構成

### 1. 📅 時系列 (Chronological)
学習した日時順にノートを保存します。
- パス: `chronological/yyyy/mm/dd/トピック名.md`
- 目的: 「いつ何を学んだか」を振り返るためのログ。
- 各月のフォルダ (`yyyy/mm/`) には、その月の学習内容一覧をまとめた `README.md` を配置します。

### 2. 🧠 知識体系 (Knowledge Base)
トピックごとに知識を体系化して保存します。
- パス: `knowledge_base/カテゴリ/トピック.md`
- 目的: 知識のネットワークを構築し、重複を排除して一つのトピックを深める。
- **動的な管理**:
    - 新しい知識は既存のファイルに**統合 (Merge)** されます。
    - フォルダ構造は学習の進度に合わせて動的に整理・統廃合されます。

#### カテゴリ例
- `Next.js/`
- `Auth/`
- `DB/`
- `General/`

## � ファイル構成 (テンプレート)

各学習ノートは以下のセクション構成で記述します。

```markdown
# [トピック名]

**学習日**: yyyy-mm-dd
**参照ドキュメント**: [ドキュメント名](URL)

## 1. 🧠 概念と前提知識（The Big Picture）
## 2. 🛠 ライブラリのアプローチとロジック
## 3. 📝 実際のコードと文脈（Code Context）
## 4. 🚫 もしこの技術を使わなかったら？（The Hard Way）
## 5. ⚖️ 技術選定とプロ視点の分析
## 6. 📖 さらなる理解のための推奨リソース
```

## 🤖 自動生成ワークフロー

`.agent/workflows/generate-learning-note.md` を使用すると、以下の処理が自動化されます。
1.  **時系列保存**: 今日の日付フォルダにノートを作成。
2.  **知識統合**: 既存の知識ベースを検索し、関連トピックがあれば追記・統合、なければ新規作成。
